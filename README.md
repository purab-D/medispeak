# MediSpeak

MediSpeak converts complex medical reports into patient-friendly explanations.

## Features

- Plain English medical report summaries
- Kannada translation support
- Eye report interpretation
- Blood report interpretation
- Child / Adult / Senior Citizen explanation modes
- Doctor discussion questions
- Key measurement extraction

## Built With

- Next.js
- TypeScript
- Gemini AI
- Tailwind CSS

## Purpose

Medical reports are often difficult for patients to understand.

MediSpeak helps bridge the gap between clinical language and everyday understanding by translating reports into simple, accessible explanations.
